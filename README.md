Phantom Crusade
Phantom Crusade is a fast-paced, 2D single-player survival game developed in the Unity engine using C#. Players must navigate a hostile environment using responsive movement controls, fend off escalating waves of intelligent enemies, and leverage item-based enhancements to progress through an intense survival game loop.

Designed with a heavy focus on arcade-style replayability, the game balances tight combat pacing with a scaling difficulty system to challenge the player's positioning and shooting accuracy.

🎮 Features
Responsive 2D Controls: Implementation of smooth WASD keyboard movement paired with a dynamic, mouse-controlled directional projectile shooting system.

Intelligent Enemy AI: Custom enemy behaviors designed to track, pursue, and swarm the player, featuring dynamic scaling mechanics that increase wave difficulty over time.

Player Progression & Enhancements: An item-based upgrade system that allows players to collect enhancements, boosting combat capabilities and survival odds.

Robust Game Loop & State Management: A fully integrated user interface featuring a main menu system, real-time player health bars, and an automated game-over/restart loop.

Balanced Pacing: Meticulously tuned combat variables, spawn rates, and damage scaling to deliver an engaging, high-replayability player experience.

🛠️ Tech Stack & Architecture
Game Engine: Unity

Language: C#

Paradigm: Component-Based Architecture (Unity Monobehaviours)

Key Systems Implemented:

Vector-based directional physics for projectiles.

Finite State Machine principles for Enemy AI tracking.

Event-driven UI updates for real-time tracking of player health states.
